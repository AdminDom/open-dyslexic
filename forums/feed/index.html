]]> http://opendyslexic.org/forums/topic/installing-opendyslexic-on-the-kindle-paperwhite-no-hacks/#post-514 <![CDATA[Installing OpenDyslexic on the Kindle Paperwhite (no hacks)]]> http://opendyslexic.org/forums/topic/installing-opendyslexic-on-the-kindle-paperwhite-no-hacks/#post-514 Thu, 17 Oct 2013 01:07:23 +0000 Abelardo Gonzalez Instructions here can be used to install OpenDyslexic, or any font you choose, on your Kindle Paperwhite without hacking the device.
You would need to change the OpenDyslexic file extension from .otf to .ttf first. Create a blank text file on the device called “USE_ALT_FONTS” and then put the OpenDyslexic files in a folder named “fonts”

Restart, and OpenDyslexic will be available as a selection.

More information:
http://blog.the-ebook-reader.com/2013/01/06/how-to-easily-add-custom-fonts-to-kindle-paperwhite-no-hacks/
